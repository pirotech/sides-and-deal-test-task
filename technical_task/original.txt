Создать две demo страницы с похожим дизайном и формой см. скрин. 
Первую страницу сделать на vue.js вторую на jquery

функционал: 

При нажатии на [+] дублируется вся строка (добавление нескольких участников)

В комбо создать несколько опций (можно статично в html) и при выборе каждой опции формировать рандомные данные в таблице,
которая показана в нижней части формы (комната 1 нежилая итд), а также данные в полях Обьект квартира и Строительные оси.
При клике на Площадь область должна сворачиватся (спойлер).

При клике на send отправляется произвольный POST AJAX запрос с данными заполненной формы + данные из окна, 
после получения ответа появляется alert(с этими же данными), либо алерт с ошибкой.
Предусмотреть верификацию формы.

При клике на load генеряться рандомные JSON данные и загружаются в форму, учесть при этом что 
"участников договора" может быть 1 или несколько (рандом), те корректировать количество полей.

Использовать следующие технологии, ничего лишнего:
Для CSS стилей использовать Sass.
Для работы с DOM на первой странице - vue.js
Для работы с DOM на второй странице - jquery
Оба js файла сделать модулями requirejs
Не использовать CDN и другие внешние ресурсы, все файлы должны идти в комплекте, страница должна работать локально (ajax запрос на 127.0.0.1)
